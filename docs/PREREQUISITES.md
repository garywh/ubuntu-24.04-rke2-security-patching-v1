# System Requirements and Dependencies

## Hardware Requirements
- **Processor:** 64-bit processor with a minimum of 2 cores
- **Memory:** Minimum 4GB of RAM, recommended 8GB
- **Storage:** At least 20GB of free disk space

## Software Requirements
- **Operating System:** Ubuntu 20.04 or 22.04 LTS
- **Docker:** Version 20.10 or later
- **Kubernetes:** Version compatible with RKE2 (check the official documentation)

## Dependencies
To ensure smooth operation, the following packages and dependencies must be installed:
- `curl`
- `wget`
- `apt-transport-https`
- `ca-certificates`

Make sure to update your system before installation with `sudo apt update && sudo apt upgrade`.  

For complete installation instructions and any additional dependencies, refer to the official documentation.
