# Troubleshooting Guide

## Common Issues and Solutions

### Issue 1: Unable to connect to the cluster
**Solution:** Ensure that the cluster configuration is correct and verify network connectivity.

### Issue 2: Node not joining the cluster
**Solution:** Check the logs for error messages and ensure the correct token is being used.

### Issue 3: Resource limits exceeded
**Solution:** Adjust the resource limits in the configuration.

### Issue 4: Pods in a CrashLoopBackOff state
**Solution:** Review pod logs for errors and adjust configurations or images as needed.
